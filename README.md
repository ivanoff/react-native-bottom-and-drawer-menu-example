# React Native Bottom and Drawer Menu Example

This is example of both Bottom menu and Drawer menu in one app.

### Install

`npm i`

### Run

#### Android

check `android/local.properties` file for `sdk.dir` property set, then

`npm run adnroid`

#### iOS

`cd ios && pod install`

`npm run ios`
